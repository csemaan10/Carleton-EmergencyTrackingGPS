# tinyGPS.ino
## Connections
- Arduino Pin 2 RX -> CAM M8C TX
- Arduino Pin 3 TX -> CAM M8C RX
- Arduino 3.3V -> CAM M8C VCC
- GND -> GND
## Extentions Required
- tinyGPSPlusPlus

# NMEAParsing.ino
## Connections
- Arduino Pin A4 -> CAM M8C SDA
- Arduino Pin A5 -> CAM M8C SCL
- Arduino 3.3V -> CAM M8C VCC
- GND -> GND
## Extentions Required
- SparkFun u-blox GNSS Arduino Library
- MicroNMEA
